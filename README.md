# BSEMC3B-SausageClash-Web (initial)

Installing on remote

Dependencies: [https://git-scm.com/](git) & [https://nodejs.org/en](node-js)

```bash
git clone https://github.com/ikeandrie/BSEMC3B-SausageClash-Web.git ./sausage-clash
cd ./sausage-clash
npm i -y
```

Development

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```
